# Django Films Application

Простое Django приложение для управления коллекцией фильмов с поддержкой импорта/экспорта данных в формате JSON.

## Функциональность

- ✅ Добавление фильмов через HTML форму
- ✅ Автоматическое сохранение данных в JSON файл
- ✅ Загрузка JSON файлов с фильмами
- ✅ Валидация загружаемых файлов
- ✅ Просмотр всех загруженных файлов
- ✅ Отображение содержимого JSON файлов
